<h1 align="center">Flags</h1>

<p align="center">
  <code>Tiny Windows tray app that shows the current keyboard layout as a flag</code>
</p>

## 🚀 What It Does

- Polls the active keyboard layout in real time and renders it as a tray icon
- Shows a matching flag for supported layouts and a language code for unsupported ones
- Includes a tray action that opens a small window with all supported flags
- Ships with an Inno Setup installer and optional auto-start with Windows

## 🏁 Supported Flags

| Code | Layout |
| --- | --- |
| `US` | United States |
| `GB` | United Kingdom |
| `UA` | Ukraine |
| `DE` | Germany |
| `FR` | France |
| `ES` | Spain |
| `IT` | Italy |
| `PL` | Poland |
| `CZ` | Czechia |
| `JP` | Japan |
